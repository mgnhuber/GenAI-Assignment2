# GenAI-Assignment2

This project focuses on building a workflow that summarizes meetings into clear action items. The primary user is a project manager who needs a fast and reliable way to turn meeting discussions into follow-up tasks. The system input will be meeting transcripts, and the output will be a list of action items along with the person or team assigned to each one.

This task is valuable to automate because project managers often spend significant time reviewing notes, identifying next steps, and tracking ownership after meetings. Automating that process can save time, reduce missed commitments, improve accountability, and help teams move from discussion to execution more consistently.
